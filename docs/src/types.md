# Supported types

* `dataclass`
* Generic types (`typing`)
    * `Dict`
    * `List`
    * `Set`
    * `FrozenSet`
    * `Tuple` / `Tuple[()]`
    * `Optional`
    * `Union`
    * `Any`
* Enum types
    * `Enum`
    * `IntEnum`
    * `Flag`
    * `IntFlag`
* Built-in types
    * `int`
    * `str`
    * `float`
    * `bool`
    * `bytes`
    * `bytearray`
    * `dict`
    * `list`
    * `set`
    * `frozenset`
    * `tuple`
* More built-in types
    * `datetime.datetime`
    * `datetime.date`
    * `datetime.time`
    * `decimal.Decimal`
    * `uuid.UUID`
